# Banner Image

Place your banner image here as `banner-image.jpg` to display it on the landing page hero section.

Recommended specifications:
- Format: JPG, PNG, or WebP
- Dimensions: 1920x1080 (or similar 16:9 aspect ratio)
- Max file size: < 500KB for optimal performance
- Subject: Fashion-related imagery that represents your brand

The image will be used as a background with an overlay, so high contrast images work best.
